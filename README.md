# Starr Surveying Website

Professional land surveying website built with Next.js 14, TypeScript, React 18, and Tailwind CSS.

## Features

- 8 Professional Pages (Home, About, Services, Resources, Service Area, Credentials, Contact, Pricing)
- Contact Form with Email Notifications
- Database Integration (Supabase)
- Mobile Responsive Design
- Professional Branding
- SEO Optimized
- Full TypeScript Type Safety

## Tech Stack

- **Framework**: Next.js 14.0.4
- **Language**: TypeScript 5.3.3
- **Frontend**: React 18.2.0, Tailwind CSS 3.3.6
- **Database**: Supabase (PostgreSQL)
- **Email**: Mailgun
- **Hosting**: Vercel

## Quick Start

### Prerequisites
- Node.js 18+
- npm 9+

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build
```bash
npm run build
npm start
```

### Type Checking
```bash
npm run type-check
```
